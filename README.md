# 🏥 Healthcare Analytics Project using SQL

## 📌 Project Overview

This project demonstrates SQL-based data analysis on a Hospital Management System database. It analyzes patient registrations, doctor workload, appointments, treatments, billing, revenue, and patient segmentation to generate meaningful business insights for healthcare organizations.

The project showcases practical SQL skills used by Data Analysts, Business Analysts, and Healthcare Analysts.

---

## 🎯 Objectives

- Analyze patient demographics and registrations
- Monitor appointment trends
- Evaluate doctor workload and specialization demand
- Analyze treatment costs and billing
- Calculate hospital revenue
- Identify VIP patients
- Perform RFM Segmentation
- Detect treatment cost outliers
- Create reporting datasets using SQL joins
